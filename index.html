Klaxon enables reporters and editors to monitor scores of sites and files on the web for newsworthy changes. Get email and Slack notifications when something changes. Sign up below, and we'll email you when it launches.

Sign up for Updates Project on GitHub

Klaxon is a Ruby on Rails app. Built and refined in the newsroom of The Marshall Project, Klaxon has provided our journalists with many news tips, giving us early warnings and valuable time to pursue stories.

We plan to release Klaxon as a free and open-source application for other newsrooms to employ. If you'd like to use Klaxon and be notified when it is released, great. If you are comfortable with Rails and want to contribute to Klaxon's development, even better. Either way, let us know so we can get in touch.
